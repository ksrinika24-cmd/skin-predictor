# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful, and inclusive environment for everyone.

## Expected Behavior

- Be respectful.
- Be constructive.
- Help others learn.
- Communicate professionally.
- Respect differing viewpoints.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Offensive language
- Discrimination
- Spam

## Enforcement

Project maintainers may remove comments or contributions that violate this Code of Conduct.

Together, we can create a positive and collaborative open-source community.
