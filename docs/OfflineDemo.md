# Offline Demonstration

## Objective

Demonstrate that Skin Predictor works without internet access.

---

## Steps

1. Turn off Wi-Fi.
2. Launch the application:

```bash
streamlit run app.py
```

3. Upload a facial image.
4. Wait for prediction.
5. View recommendations.
6. Generate PDF report.

---

## Expected Result

The application performs all operations locally without any cloud communication.

---

## CPU Runtime

Inference Engine:

- ONNX Runtime

Execution Device:

- CPU

Internet Required:

No
