# User Manual

## Introduction

Skin Predictor is a Python application that predicts skin conditions from images.

## Requirements

- Python 3.10+
- Install dependencies from requirements.txt

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

## Usage

1. Start the application.
2. Upload a skin image.
3. Click **Predict**.
4. View the prediction.

## Troubleshooting

- Ensure all dependencies are installed.
- Verify the image format is supported.
- Check that Python is installed correctly.
