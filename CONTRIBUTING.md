# Contributing to Skin Predictor

Thank you for your interest in contributing.

## Getting Started

1. Fork the repository.

2. Create a new branch.

```bash
git checkout -b feature-name
```

3. Make your changes.

4. Run tests.

```bash
pytest
```

5. Commit using semantic commit messages.

Example:

```
feat: add recommendation engine
```

6. Push your branch.

7. Open a Merge Request.

---

## Coding Guidelines

- Follow PEP 8
- Write meaningful comments
- Add unit tests for new features
- Keep functions modular
- Ensure code passes CI checks

---

## Reporting Issues

Please include:

- Operating System
- Python Version
- Error message
- Steps to reproduce
