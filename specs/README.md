# Specifications

This directory contains feature-level specifications, task plans, and quality
checklists for Skin Predictor.

Each feature directory should include:

- `spec.md`: Product and engineering specification.
- `tasks.md`: Implementation and verification task plan.
- `checklist.md`: Acceptance and release readiness checklist.

Specifications must be kept current with the implementation. Compliance-only
updates should document process and repository structure without changing
application behavior.
